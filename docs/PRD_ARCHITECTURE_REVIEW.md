## 0. 完整产品定义

### 产品概述与核心价值
- **产品名称**: MindFlow
- **定位**: WPS生态下的下一代 AI 思维导图与知识工作台
- **核心差异化**: 节点内联层级折叠技术（允许在单一节点内部无限层级折叠/展开，而非只创建外部子节点），实现"一屏看清全局，点击深入细节"。
- **三大支柱**:
  - 功能创新: 内联层级 + 八大视图（画布、大纲、甘特图、看板、日历、幻灯片、白板、知识图谱）
  - 最高安全: 端到端加密（信封加密、HSM）、五步可视化解密仪式、企业级数据隔离
  - 立体体验: 轻量3D景深、粒子特效、专业可信的 UI 设计语言

### 节点与内联层级核心交互
- 节点外部框体（卡片）: 支持圆角矩形、菱形、星形、旗帜等多种形状，可带图标、进度条、优先级标签、自定义标签、附件、备注。
- 内部行（InnerLine）模型: 每行可缩进无限层级，行首有折叠三角，行尾可显示任务复选框。支持拖拽排序、缩进/减少缩进。
- 节点连接点: 整个节点边框都是可交互热点区，鼠标悬浮到边框变为十字准星，拖拽即可创建连线。连线动态吸附到目标节点最近边，支持直线/折线/贝塞尔曲线，连线可添加标签，颜色、粗细、箭头等样式可调。
- 编辑交互: 双击节点编辑文本（富文本），右键节点上下文菜单（根据节点类型动态变化），框选批量操作，键盘快捷键（Tab/Shift+Tab缩进，Ctrl+[折叠等）。
- 撤销/重做: 无限制步数，支持事务合并。

### 八大视图生态系统
- 所有视图共享同一份底层数据（内联层级树+CRDT协同数据），任一视图的修改实时同步到其他视图。
- **画布视图**: 默认主视图，无限画布，节点卡片展示内联行，连线智能避障，小地图导航。
- **大纲视图**: 树形文本列表，外部节点带彩色标记，内部行纯缩进，支持搜索、虚拟滚动。
- **甘特图视图**: 将带日期的行映射为甘特条，外部节点为汇总，子节点为子任务。支持依赖、今日线、关键路径、计划vs实际对比。
- **看板视图**: 按状态分列，卡片拖拽，WIP限制，列自定义。
- **日历视图**: 月/周/日切换，显示任务事件，拖拽改期。
- **幻灯片视图**: 一级节点为幻灯片页面，内联行为逐条动画，支持折叠控制演示节奏，演讲者视图。
- **白板视图**: 自由画布，画笔、形状、便签，导图节点可拖入白板，手绘可转为导图节点。
- **知识图谱视图**: 力导向布局，2D/3D模式，节点大小表示关联度，折叠行以隐含节点表示，支持过滤和高亮路径。

### 任务功能与跨视图联动
- 任意内部行可标记为任务，设置状态、指派人、截止日期、优先级、预计工时、依赖关系。
- 在甘特图、看板、日历、大纲中自动呈现并同步状态。任务完成时，甘特条自动更新进度，看板卡片可拖拽改变状态。

### 多方实时协作
- 算法: 基于CRDT（Yjs）的行级无锁协同，支持离线编辑自动合并。
- 多光标/头像显示，他人编辑的行旁有彩色标记。
- 评论与@提及，集成通知。
- 冲突处理: 语义冲突自动标记，提供可视化对比和手动解决界面。
- 权限: 节点/行级权限，分享链接可设密码、有效期、禁止下载。

### 最高安全架构
- 传输: 全链路TLS 1.3，API请求HMAC签名防重放。
- 存储: 信封加密（文件密钥FK用租户KEK加密，KEK由HSM管理），客户端解密（五步解密仪式）。
- 密钥管理: HSM硬件安全模块，双人审批，操作审计，明文绝不落盘。
- 企业策略: 强制二次验证、设备远程撤销、屏幕水印、审计日志。

### AI 浮动控制台 (MindFlow AI)
- 形态: 独立浮动窗口，可拖拽/缩放/最小化，非模态。
- 交互流程: 用户输入 → AI解析并生成结构预览 → 用户可编辑预览 → 应用到画布。
- 支持连续对话迭代、图形类型选择面板、智能纠错。
- 底层: 结构化Prompt生成完整JSON → 渲染为可编辑原生导图对象。

### UI/UX 设计规范摘要
- 设计语言: 深空蓝灰+珍珠白，信号青强调，警示琥珀。
- 立体化: 节点选中上浮4px+外发光；折叠动画有吸入感；解密粒子3D保险库动效。
- 动效参数: 折叠展开300ms cubic-bezier(0.4,0.0,0.2,1)，图标弹性200ms，弹窗淡入250ms。
- 主题: 完整深色/浅色模式。

### 数据与部署（当前状态）
- 前端: Vue 3 + TypeScript, ECS (47.86.31.53), Nginx
- 后端: NestJS, PM2, 端口3000
- 数据库: Supabase (PostgreSQL), 绕过TLS SNI问题
- RLS策略: 已启用
- DNS: Cloudflare zjhdml.online
- 仓库: GitHub (ZJHWY/mindflow)

## 1. 严格审查：逻辑不自洽、遗漏或错误

### 遗漏 1: 内部行的连线与关联
当前连线功能仅存在于"节点"（外部框体）之间。但 MindFlow 的核心是"内联层级"，如果用户想在"会议纪要"节点下的两个不同行之间建立关联（例如，"讨论点A"依赖于"数据报告B"），目前无法实现。**这是核心功能的缺失。**

**建议**: 为内部行增加"行内引用/关联"功能。可通过拖拽行尾的连接点或使用 `[[行文本]]` 语法创建双向链接。这些链接应在画布和白板视图中有特殊的视觉表现。

### 遗漏 2: 模板与示例库的引导不足
虽提到"3步交互式引导"，但未提及可复用的结构化模板库。

**建议**: 将模板库作为新建文件时的首要选择，并允许用户将个人作品保存为模板。

### 遗漏 3: 全局搜索与替换
缺少**跨文件、跨项目的全局搜索**能力。

**建议**: 在顶部导航栏增加全局搜索入口，支持搜索节点文本、标签、备注、附件名称。搜索结果需高亮定位到具体视图和节点。

### 不自洽: "多视图数据互通"与某些视图的冲突
- 甘特图/日历依赖时间字段，无日期的节点如何处理？
- 看板依赖状态字段，默认状态是什么？

**建议**: 甘特图/日历增加"未安排任务"收纳箱，看板增加"未分类"列，确保所有内容都有归属。

### 遗漏 4: 多人协作下的"视图状态"同步
A 在看板视图拖拽卡片改变任务状态，B 的画布上该节点的标签/图标应同步更新。

**建议**: 需要跨视图的实时状态同步机制。

## 3. 差异化建议：形成绝对优势的杀手级功能

### 1. "时间回溯"功能
可视化时间轴滑块，拖动滑块时整个画布像时光倒流一样动态重构。**复现整个思考过程**，对复盘和学习价值巨大。
- *竞品弱点*: XMind, 飞书只有基于列表的版本历史，无法直观看到过程。

### 2. "演示者视图"2.0（带思维引导）
AI 为每一页生成演讲备注、关键提问和过渡话术。**"虚拟听众模式"**: AI 实时通过耳机或字幕给予互动反馈。
- *竞品弱点*: 当前所有演示工具都只是静态放映。

### 3. "智能重构"引擎
框选混乱节点 → 右键"智能重构" → AI 分析内在逻辑 → 自动重排成1-3种逻辑结构供选择。
- *竞品弱点*: 竞品只有简单自动排序，无基于语义的逻辑重构。

### 4. "白板-文档"双向穿透
魔法套索圈选手绘/便签 → 一键转为结构化思维导图节点。导图分支 → 一键"炸开"成白板自由元素。
- *竞品弱点*: Miro有自由，XMind有逻辑，两者割裂。MindFlow 让它们无缝流转。

### 5. "协同沙盒"模式
团队成员临时进入沙盒模式，任意修改不影响原始导图。操作完成后以"变更提案"提交，负责人选择性接受/拒绝。
- *竞品弱点*: 其他工具的"建议模式"通常逐行文字，而非整个结构更改尝试。

## 4. AI 策略深化：从被动响应到主动顾问

### 上下文感知的洞察卡片 (Insight Cards)
- **检测逻辑矛盾**: 流程图分支指向同一节点 → 提示修正
- **检测信息孤岛**: 无连线节点 → 建议自动连接
- **任务协助**: 逾期任务 → 提示通知负责人/改状态

### 画布内主动可视化建议
- 选中节点周围以30%透明度浮现2-3个AI建议的"下一步"节点
- 连线徘徊时，自动变为虚线并悬浮显示预览效果

## 6. 技术可行性速评

### 高风险 1: 10,000+节点内联层级与复杂连线的流畅渲染
- **风险**: 传统 DOM 无法支持，Canvas/SVG 混合渲染架构复杂度极高
- **替代方案**: 设定画布范围上限或节点数阈值，超出后引导使用大纲视图。借鉴游戏引擎 ECS 架构和 Web Worker 处理布局计算

### 高风险 2: CRDT 行级实时协同 + 复杂语义冲突
- **风险**: 复杂树形操作（移动、缩进、连同子节点操作）转化为 CRDT 操作极易出现数据不一致
- **替代方案**: 多数操作用 CRDT 自动合并；高风险结构性操作采用"操作锁"机制或"待确认"状态标记

### 高风险 3: "智能重构"和"主动AI建议"的实时性与准确性
- **风险**: 需要低延迟（<2秒），否则打断心流；AI 容易给出"废话建议"失去信任
- **替代方案**: 先做好"被动但精准"的 AI 功能；主动功能从简单的"模式识别"开始
